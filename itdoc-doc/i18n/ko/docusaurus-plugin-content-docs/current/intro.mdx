# itdoc이 뭐죠?

신뢰할 수 있는 테스트 기반 API 문서를 생성하세요 – Node.js 테스트 코드만으로!

## 기능

- 🧪 테스트 기반 API 문서 자동 생성
- 📄 OpenAPI, Markdown, HTML 출력 지원
- 🚫 테스트가 실패하면 문서를 생성하지 않음 – 정확성 보장
- 🔧 기존 테스트 스위트와 쉽게 통합 가능
- 🧩 프레임워크 무관 (Express, Fastify 등)
- 🤖 GPT 기반 테스트 케이스 생성으로 더 빠른 문서화

## 프로젝트의 목표

이 프로젝트의 주요 목표는 `JavaScript`로 작성된 `RESTful 웹 서비스`를 신뢰성 있게 문서화하는 것입니다. 일반적인 JSON 또는 JSDoc 기반 API 문서화 방식과 달리, `itdoc`은 테스트 코드에서 요청/응답 예제를 직접 추출합니다. 테스트가 통과해야만 문서가 생성되므로, 항상 최신이며 검증된 API 정보를 배포할 수 있습니다.

`itdoc`은 여러분이 작성한 설명과 테스트 결과를 결합해 문서를 생성합니다.

테스트 기반 문서는 API의 실제 동작을 항상 반영합니다. 테스트가 통과하기만 하면, 문서를 OpenAPI 스펙, Markdown, HTML 등 다양한 형식으로 내보내어 원하는 방식으로 배포할 수 있습니다.
